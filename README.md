# 🤖 Crypto Alert Bot v2

Confluence-бот для крипто-скальпинга: следит за топ-500 альтами через 8 детекторов, отправляет алерты в Telegram только когда несколько сигналов совпадают одновременно.

## Что внутри

**Источники данных:**

- Binance Spot/Futures API (свечи, стакан, OI, funding)
- Binance Futures WebSocket (ликвидации в реальном времени)
- CoinGecko (список монет, маркеткап)
- Etherscan (on-chain netflow для ERC-20)

**8 детекторов** — каждый независимо оценивает монету и присваивает сигналу вес:

| Детектор         | Что ловит                                                 |
| ---------------- | --------------------------------------------------------- |
| `volume_spike`   | Объём x5/x7/x10 от baseline для этого часа недели         |
| `price_move_atr` | Движение 1.5/2/3 ATR — адаптивно под волатильность монеты |
| `breakout`       | Пробой хая/лоя 4ч / 24ч / недели с подтверждением объёмом |
| `orderbook`      | Крупные стены ($500k+), дисбаланс bid/ask                 |
| `open_interest`  | ±3/5/10% OI за 15 минут с интерпретацией по цене          |
| `funding`        | Экстремальный funding (±0.05% / ±0.1%)                    |
| `liquidations`   | Каскады $1M+ / $5M+ / $10M+ за 5 минут                    |
| `netflow`        | Крупные переводы ERC-20 на/с бирж                         |

**Confluence Engine** собирает сигналы в окне 15 минут и решает:

- **👀 watch** (score ≥ 2.0, ≥1 детектор) — к сведению
- **⚡ signal** (score ≥ 4.0, ≥2 детектора) — торговый сигнал
- **🚨 strong** (score ≥ 6.5, ≥3 детектора) — сильная конвергенция

## Установка — 5 минут

### 1. Настрой Telegram

- Создай бота через [@BotFather](https://t.me/BotFather), получи токен
- Создай канал, добавь бота как админа
- Узнай chat_id канала: пошли сообщение в канал → открой `https://api.telegram.org/bot<TOKEN>/getUpdates`

### 2. (Опционально) Etherscan API key

Для on-chain netflow. Бесплатно на [etherscan.io/myapikey](https://etherscan.io/myapikey).
Если не настроишь — netflow выключится, остальное работает.

### 3. Заполни `config.yaml`

```yaml
telegram:
  bot_token: "123456789:ABC..."
  chat_id: "-1001234567890"

detectors:
  netflow:
    etherscan_api_key: "ABC123..." # опционально
```

### 4. Preflight check

```bash
pip install -r requirements.txt
python3 preflight.py
```

Покажет что работает, что нет.

### 5. Запуск

```bash
python3 main.py
```

## Команды в Telegram

Пиши боту прямо в канал:

```
/status       — uptime, universe, активные сигналы, текущие пороги
/stats        — статистика алертов за 24 часа
/top          — топ монет по активности сигналов прямо сейчас
/pause        — приостановить алерты
/resume       — возобновить
/threshold signal 5.0    — изменить порог (watch/signal/strong)
/cooldown signal 45      — изменить cooldown в минутах
/mute DOGE 60            — замьютить монету на 60 мин
/unmute DOGE             — размьютить
/test                    — отправить тестовый алерт
/help                    — справка
```

Все изменения применяются **без перезапуска**.

## Анализ эффективности

Бот ведёт журнал всех отправленных алертов в SQLite (`alerts.db`).

Раз в сутки запускай:

```bash
python3 report.py
```

Это:

- Проверяет цену через 1/4/24ч после каждого алерта
- Считает % точности по tier и по детекторам
- Шлёт сводку в канал

Через 2-3 недели накопится статистика — можно будет калибровать пороги по реальным данным: выкинуть шумные детекторы, поднять вес точных.

В cron:

```
0 12 * * * cd /path/to/bot && python3 report.py
```

docker compose run reporter

## Добавление людей в канал

Чисто Telegram — ничего в коде:

1. Настройки канала → пригласить → получить ссылку
2. Разослать
3. Если хочешь чтобы только ты управлял ботом: в `commands.py` команды принимаются только из канала, указанного в `chat_id`. Остальные пользователи могут читать алерты, но не управлять.

## Калибровка для разных стилей

Сейчас конфиг настроен под **скальпинг топ-500**. Если нужен другой стиль:

**Свинг-трейдинг (дни-недели):**

```yaml
scan_interval_seconds: 300 # раз в 5 мин хватит
detectors:
  price_move_atr:
    atr_period_minutes: 1440 # ATR на дневных свечах
  breakout:
    lookback_hours: [24, 168, 720] # сутки, неделя, месяц
confluence:
  window_minutes: 240 # 4 часа на конвергенцию
```

**Только мемкоины / новые монеты:**

```yaml
filters:
  min_market_cap_usd: 1000000 # 1M — микрокапы
  min_volume_24h_usd: 500000
  min_age_days: 1 # совсем свежие
```

Отключи `open_interest` и `funding` — для мемкоинов на спотах это не работает.

**Только твои монеты:**

```yaml
filters:
  whitelist: ["BTC", "ETH", "SOL", "SUI"]
```

## Частые проблемы

**"Binance блокирует"** → ты из US/GB/подсанкционной страны. Запускай через VPN или на VPS в Европе.

**"Мало алертов"** → подожди 30-60 минут для накопления baseline. Если и потом тишина — уменьши пороги через `/threshold watch 1.5`.

**"Слишком много алертов"** → `/threshold signal 5.0`, `/cooldown signal 60`.

**"Netflow ничего не пишет"** → это нормально, крупные переводы бывают редко. Через сутки должно хотя бы пару раз сработать на топ-токенах (LINK, UNI, PEPE и т.п.).

**CoinGecko 429** → увеличь `scan_interval_seconds` до 60 или 120.
